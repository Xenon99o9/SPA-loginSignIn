# SPA-loginSignIn
Projet d'apprentissage création d'un login signIn

__Objectif du TP :__

Créer une mini‑application SPA (Single Page Application) sans framework, contenant :

- une page Connexion
- une page Inscription
- une page Accueil (après connexion)

L’application utilise une seule page HTML, et JavaScript gère l’affichage dynamique des écrans.
Une fausse API simule les requêtes (inscription / connexion).
